# Skin Tone & Lighting Analyzer

A simple Streamlit app that analyzes facial lighting conditions and skin tone uniformity using OpenCV.

## Features
- Detects face from uploaded selfie
- Calculates lighting score
- Calculates skin tone variation
- Gives human-friendly feedback

## Tech Stack
- Python
- Streamlit
- OpenCV
- NumPy
- Pillow

## How to Run
```bash
pip install streamlit opencv-python numpy pillow
streamlit run app.py
```
